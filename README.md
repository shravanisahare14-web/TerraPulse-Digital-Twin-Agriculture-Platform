## 🚀 Overview

TerraPulse is a Digital Twin Agriculture Intelligence Platform designed to simulate, monitor, and analyze agricultural environments through real-time environmental intelligence.

The platform combines IoT architecture, environmental simulation, smart irrigation monitoring, dynamic risk assessment, and interactive analytics to demonstrate how modern technology can improve agricultural decision-making.

Rather than relying on static datasets, TerraPulse continuously generates realistic agricultural telemetry using a Digital Twin Engine and transforms that data into actionable insights.

# ▶️ Running The Project

## 1. Clone Repository

```bash
git clone https://github.com/shravanisahare14-web/TerraPulse-Digital-Twin-Agriculture-Platform.git
```

---

## 2. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 3. Start Digital Twin

```bash
python simulation/generate_data.py
```

---

## 4. Launch Dashboard

```bash
cd dashboard
python -m http.server 8000
```

---

## 5. Open Browser

```text
http://localhost:8000
```
